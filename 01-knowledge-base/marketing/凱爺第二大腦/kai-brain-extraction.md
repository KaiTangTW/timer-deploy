## A. 凱爺的核心品牌哲學

### 世界觀：品牌是一套宇宙觀，不是一個 LOGO

1. **從名字到品牌的三段進化**：名字 → 名牌（知名度）→ 品牌（指名度）。唯有擁有指名度的品牌，才能擁有超額利潤。

2. **Value Up，不是 Cost Down**：品牌人生不該只有 cost down 策略，而更該做的是 value up。品牌定位就是為了找到獨特價值（USP），讓消費者選擇你不是因為你最便宜，而是因為你最獨特，非你不可。

3. **品牌是結果論，也是過程論**：品牌不只是最後消費者看到的成果，建立品牌的討論過程本身就在建立內部共識，二者缺一不可。

4. **品牌即隱形人**：品牌本身是看不見的，需要一層層的行銷包裝讓它現形。但衣服不是越多越好，亂搭亂配會讓定位模糊。

5. **投影機理論**：品牌就像會議室中央的投影機，所有人從不同角度看它，初期不可能一致，所以需要一套系統來建立共識。

6. **品牌四環理論**：品牌訊息從核心幹部（第一環）→ 企業內部非品牌人員（第二環）→ 外部協力夥伴（第三環）→ 消費者（第四環），每經一環訊息就會失真，所以需要360系統來確保一致性。

7. **幫助台灣中小企業就是回饋社會**：台灣98%的從業人員在中小企業，幫助中小企業永續經營，就是讓人民安居樂業。

8. **加值是品牌的核心動作**：在經營品牌的過程中，每個步驟都要問「這個做法對消費者的價值是什麼？能提高願付價格嗎？」

9. **第二曲線思維**：在首次成長期就該為下一次成長做準備，唯有掌握這個概念的企業才能基業長青。

### 凱爺反對的事

- 低價策略作為品牌核心（CP值不該是品牌核心）
- 暴力折扣（拿未來的血暖現在的自己）
- 對客戶想像失真、說一套做一套
- 孤決（主理人一個人的喜惡決定企業生死）
- 用形容詞做品牌核心（時尚、頂級、年輕...無法被證明）

## B. 凱爺的語氣 DNA

### 句式結構

1. **先拋問題再解答**：「你是不是覺得只有B2C企業才需要做品牌？其實...」「大家有沒有想過品牌訊息是怎麼接觸到我們的？」
2. **先講故事再總結**：每個理論都配一個真實案例或親身經歷，案例之後才點出道理。
3. **口語化長句**：不怕句子長，用逗號串接思路，像在課堂上對著學生講話。
4. **反問句收尾**：「如果你是業者，你會選擇什麼樣的品牌呢？」「你們真的覺得鴻海白牌手機能幹掉蘋果？」
5. **自嘲式幽默**：「一時之間我還無法適應」「我想說這有什麼好問的」「國色天香，她害羞了！」

### 口頭禪與慣用語

- 「也因此」（極高頻率的因果連接詞）
- 「舉例：」（幾乎每段理論後必接）
- 「後續我們會教導大家...」「後續我們也會在...詳細說明唷」
- 「大家辛苦了」（章節結尾的溫暖收束）
- 「小提醒：」（每章結尾的便利貼作業指引）
- 「唷」（句尾語助詞，製造親切感）
- 「啦」（「會卡卡的啦」「你也可以...唷」）
- 「縱使」（偏好用古雅的連接詞）
- 「誠如」（正式但不僵硬）

### 比喻與修辭手法

- **古典引用**：桃花扇「眼看他起朱樓，眼看他宴賓客，眼看他樓塌了」
- **鼎的三隻腳**：品牌核心三個字詞要像鼎的三隻腳一樣保持距離
- **投影機**：品牌就是投影機，大家從不同角度看
- **隱形人**：品牌是隱形人，行銷是它的衣服
- **比手畫腳遊戲**：品牌四環的訊息失真
- **蚯蚓的肚子**：通路越多，表格縱軸越長
- **溫度計**：客情從生→溫→熱→熟→鐵
- **地圖**：品牌宇宙觀是一張按圖索驥的地圖

### 語言風格特徵

- **正式度**：介於教授與大哥之間。有學術框架，但用口語舉例。不會說「本研究發現」，但也不會太隨便。
- **情感溫度**：溫暖但不煽情。對學生有耐心，會說「別擔心」「不要緊張」「如果覺得還無法果斷下手，別擔心」。
- **幽默風格**：冷幽默、自嘲型。「抱歉，是誠意邀請他們接受訪談」（講完嚴刑拷打後）。棒球棒那段汽車用品店的暗示也很典型。
- **說教但不高高在上**：會分享自己的失敗和學習過程，用「我一路從大學到研究所」拉近距離。
- **善用數字強化論點**：蘋果毛利46.58% vs 小米14.6%、新客成本是舊客5倍、視覺87%等。

### 解釋複雜概念的方式

1. **先給框架縮寫**：SME（感性利益）、CRISP（理性利益）、AEIOU（觀察法）
2. **再用生活案例拆解**：飯店「歡迎回家」、DoubleTree 餅乾、鬆餅店觀察女孩換位
3. **最後給作業**：「請把你的品牌核心寫在便利貼上」
4. **循序漸進**：每個新概念都先回顧前面教過的，再推進

### 1. New STP 理論（全新 STP 2.0）

傳統 STP 的市場區隔過度簡化消費者，凱爺提出：

- **S = Stage of Life（生命階段）+ Situation（情境）**
  - 9大生命階段：單身→情侶→同居→多元家庭→公婆同住→空巢期→獨居老人（可跳躍、可回退）
  - 生命階段 × 職位表（學生/小資/上班族/主管/企業主/職業婦女/家庭主婦/退休族）= 可支配所得 + 消費心態
  - 情境可天然（天氣、節氣）或人為（雙11、促銷）
- **T = Targeting**：目標市場不僅要有需求，還必須是 Affordable（可負擔）的人
- **P = Positioning**：持續在目標市場心中塑造獨一無二、具差異化的品牌形象

### 2. 360品牌定位系統（八大環節）

按順序：

| # | 環節 | 關鍵內容 |
|---|------|----------|
| 1 | **品牌核心** | 3個核心字詞（2理性+1感性），來自企業獨有資源與競爭優勢。像鼎的三隻腳要保持距離。避免用形容詞、避免CP值 |
| 2 | **品牌承諾** | 「品牌為誰做什麼」= WHO + WHAT（do/benefit/where）。一句話，正面表述，可一口氣說完 |
| 3 | **品牌象徵（五感）** | 視覺87%、聽覺7%、觸覺3%、嗅覺2%、味覺1%。嗅覺是唯一不需大腦理性分辨就能產生感性反應的形式 |
| 4 | **權威基礎金字塔** | 四層：第三方權威機構 → 專家達人 → 名人藝人KOL → 素人。上窄下寬，上層信任權重高，下層適合大量活用 |
| 5 | **消費者感性利益 SME** | **S**enses五感 + **M**aslow馬斯洛8層需求（生理/安全/社交/尊重/認知/審美/自我實現/超越）+ **E**motion情緒連結與共感同理 |
| 6 | **消費者理性利益 CRISP** | **C**ost交易成本 + **R**isk風險 + **I**nvisible Cost隱形成本（時間/勞力/精神/配套）+ **S**ervice服務 + **P**erformance性能 |
| 7 | **品牌個性** | 基於榮格原型12型（革命者/魔術師/英雄/情人/小丑/凡夫俗子/慈善家/統治者/創造者/天真者/智者/探險家）。三種應對方式：上對下/下對上/平起平坐 |
| 8 | **消費者印象** | 功能性印象 + 象徵性印象 + 經驗性印象。用AEIOU情境描述法 |

**360系統診斷法**：
- 筆直放中間 = 左感性/右理性，檢查是否偏向一邊
- 筆橫放 = 上方品牌內部金三角（核心+象徵+權威）/ 下方品牌外部金三角（承諾+個性+印象）/ 中間消費者利益

### 3. 消費者分析方法

- **量化研究**：問卷調查、滿意度調查
- **質化研究**：深度訪談、焦點團體訪談
- **觀察研究法**：AEIOU（Activities活動 / Environment環境 / Interactions互動 / Objects物件 / Users使用者）
- **深度洞察法（蹲點）**：與消費者共同生活
- **核心心法**：See him. Ask him. Be him.（觀察他、詢問他、成為他）

- 一周7天 × 每天7個時段

- 三張透明紙疊加 = 找到重點時間與離峰時間

### 5. 體驗設計十大階段

```
認知 → 探索 → 試用 → 購買 → 使用 → 保持 → 升級 → 替換 → 評價 → 回饋
```

關鍵概念：
- **認知**：有意識需求 vs 無意識需求；實際狀態需求 vs 理想狀態需求
- **探索**：心智梯子最多7個品牌，搶前3名
- **試用**：剝奪感 + 困難度
- **購買**：降低交易成本、風險、提供選購建議
- **保持**：迎新 + 顧舊，會員經營心法（錢給不了的才是頂級會員要的：臉面/專屬/社交/特權）
- **替換**：客訴三步驟 = 面對 → 處理 → 優化。心法：換位思考，以人為本
- **評價**：易於分享的三性 = 物質性 + 文化性 + 社交性
- **回饋**：品牌要調整個性讓消費者感到被聆聽

記憶與情緒的連結：
- 高於平均值的好 → 旅程初期與結尾
- 最高值的好 → 旅程結束之前（驚艷感動時刻）
- 高度落差的好 → 客訴逆轉勝

### 6. 客情溫度計

```
生 → 溫 → 熱 → 熟 → 鐵
```

| 階段 | 消費者給出的 | 品牌得到的 |
|------|-------------|-----------|
| 生→溫 | 時間 | 獲利（超額利潤）|
| 溫→熱 | 金錢 | 名氣（知名度→指名度）|
| 熟→鐵 | 人生 | 信仰（品牌初衷、願景、人品相合）|

### 7. 營業額方程式

```
外部流量 × 外部轉換率 = 內部來客數 × 內部轉換率 × 客單價 = 營業額
```

來客成本方程式：
```
新客 × 新客取得成本 + 舊客 × 舊客回購成本（新客成本×0.2）= 品牌行銷總成本
```

新舊客五比一法則：獲得新客成本是維持舊客的5倍。

### 8. 全年度品牌行銷行事曆

#### 天時（四種類型）
1. **節慶**：有放假的 + 沒放假但有文化意義的
2. **節氣**：24節氣各有對應產業商機（詳細列出每個節氣的特色與商機）
3. **人造節**：購物節（雙11/黑五/百貨周年慶）+ 國際XX日/世界OO日
4. **品牌日**：品牌自創的特殊日子
5. **新品上市**（補充項）

#### 地利
- **通路**：實體（量販/便利商店/藥妝/百貨/專賣店）+ 虛擬（官網/購物平台）+ 特殊（團購/直播/直銷）
- **地利之便（便宜）**：三限兩滿
  - **三限**：限時（建議10-14天）、限地、限量
  - **兩滿**：滿減（目的性折扣，不可暴力打折）、滿贈（模糊化價格分析、提升品牌價值、創造新鮮感）

#### 人和（六大管道）
1. **社群工具**（自媒體）：品牌文/知識文/販促文/生活文，8:2比例（品牌+知識 vs 販促）
2. **媒體採購**（付費它媒體）：傳統廣告/數位廣告/戶外廣告
3. **會員經營**：迎新 + 顧舊。會員導向 vs 情境導向
4. **消費者活動**：Event，面對面互動
5. **媒體公關**（免費它媒體）：品牌高度內容 + 販促消息稿。代言人預算法則：一加一、一比三。合約建議改為1+1+4
6. **異業合作**：品牌對品牌/品牌對IP/品牌對通路

### 10. 情境九宮格

用於擬定品牌關鍵訊息：

```
┌──────┬──────┬──────┐
│ 彼時  │  事  │ 此時  │
├──────┼──────┼──────┤
│ 彼地  │ 商品 │ 此地  │
├──────┼──────┼──────┤
│更好的人│ 通路 │當下的人│
└──────┴──────┴──────┘
```

左三格 = 未來（購買後），中間 = 品牌商品+通路+關鍵訊息，右三格 = 現在（購買前）

### 12. 品牌健康自我檢測（六症狀）

1. 賣不出去 → 源頭問題，需重做 New STP
2. 差不多先生 → 無差異化，需找 USP
3. 窮忙族 → 有量無利，毛利被壓縮
4. 新客轉換率低 → 體驗設計前三階段出問題
5. 會員忠誠度低 → 360系統未落實，感性理性利益缺漏
6. 內部溝通成本過高 → 需要360系統建立共識

### 13. 凱爺的三個理論模型

| 理論 | 核心概念 |
|------|----------|
| 投影機理論 | 品牌是中央投影機，不同角度看都不同，需要系統建立共識 |
| 隱形人理論 | 品牌本身隱形，靠行銷包裝現形，但不可亂搭 |
| 品牌四環理論 | 訊息從核心到消費者經過四環傳遞會失真，需要系統確保一致性 |

### 14. 經營品牌七大挑戰

1. 定位問題（只會 cost down）
2. 對客戶的想像錯誤
3. 言行不一（定位與執行脫節）
4. 訊息紛雜（垂直+水平訊息失真）
5. 未能洞察消費者需求改變
6. 未能與時俱進（科技演進）
7. 其它外部危機

### 方法論之間的連結關係

```
New STP（找到對的人和需求）
    ↓
360品牌定位系統（建立品牌內涵）
    ↓
品牌故事速成法（將定位變成故事）
    ↓
全年度品牌行銷行事曆（落地執行）
    ├── 天時（節慶/節氣/人造節/品牌日）
    ├── 地利（通路 + 三限兩滿）
    └── 人和（六大管道）
         ↓
    檔期企劃六要素（每檔活動規劃）
         ↓
    情境九宮格（擬定關鍵訊息）

同時，消費者端：
目標市場周行事曆 → 體驗設計十大階段 → 客情溫度計 → 營業額方程式（量化追蹤）
```

品牌自我檢測可在任何階段執行，用六症狀 + 營業額方程式 + 360系統左右上下診斷法交叉檢核。
